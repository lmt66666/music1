# music

# 下载须知:
本项目api的是 Binaryify 后端api 地址: https://github.com/Binaryify/NeteaseCloudMusicApi 

## 下载使用 
```
git clone https://github.com/Binaryify/NeteaseCloudMusicApi
```
## 开启后端服务
```$xslt
node app.js
```

## 下载本项目
```$xslt
git clone https://github.com/coderps123/music.git
```
## 运行
```$xslt
npm install
npm run serve
```
